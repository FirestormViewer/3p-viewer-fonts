# 3p-viewer-fonts

Viewer fonts for the Emoji Project contribution from Kitty  Catznip (kitty@catznip.com)

## Building
Use autobuild configure/build/package as normal. Note that for official builds we use platform=common. 

The version is determined by the `fonts_version` variable, which can be set in the environment. If not set, it defaults to `1.0.0`.
For official builds FS automated builds, this is expected to be extracted from the tag of the current main branch, where the format can be vN.N.N or N.N.N.
